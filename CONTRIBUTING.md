# 🤝 Contributing to ExoLogica AI

We welcome contributions from the global scientific community! This project is open for verification, improvement, and collaboration.

**(Мы приветствуем вклад от мирового научного сообщества! Этот проект открыт для проверки, улучшения и сотрудничества.)**

## How to Contribute / Как Внести Вклад

### 1. Data Verification / Проверка Данных
- Check our datasets for errors or inconsistencies
- Suggest additional exoplanet data sources
- Verify density calculations and PRI values

### 2. Code Improvements / Улучшения Кода
- Optimize analysis algorithms
- Add new visualization features
- Improve documentation
- Fix bugs or edge cases

### 3. Scientific Validation / Научная Валидация
- Test the ExoLogica Law with your own datasets
- Compare with other habitability models
- Suggest physical improvements to the model
- Provide peer review feedback

## Getting Started / Начало Работы

1. **Fork the repository** / Форкните репозиторий
2. **Create a branch** / Создайте ветку (`git checkout -b feature/amazing-feature`)
3. **Make your changes** / Внесите изменения
4. **Test thoroughly** / Протестируйте тщательно
5. **Commit your changes** / Закоммитьте изменения (`git commit -m 'Add amazing feature'`)
6. **Push to the branch** / Отправьте в ветку (`git push origin feature/amazing-feature`)
7. **Open a Pull Request** / Откройте Pull Request

## Code Style / Стиль Кода
- Follow PEP 8 guidelines for Python code
- Include docstrings for all functions
- Write clear comments in English
- Use meaningful variable names

## Reporting Issues / Сообщение о Проблемах
If you find bugs or have suggestions:
- Use the GitHub Issues tab
- Provide detailed description
- Include steps to reproduce
- Attach relevant data files

## Scientific Collaboration / Научное Сотрудничество
For academic collaborations:
- Contact us via GitHub Discussions
- Share your research findings
- Propose joint publications
- Suggest conference presentations
