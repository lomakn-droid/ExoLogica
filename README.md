# 🌌 ExoLogica AI: The Critical Density Law of Habitability
# (ExoLogica AI: Закон Критической Плотности Обитаемости)

> **Abstract:** We present a new fundamental principle of astrobiology — the **ExoLogica Law** (Critical Density Window Principle). Based on an analysis of 9,800+ exoplanets using our hybrid AI engine, we demonstrate that habitability is strictly constrained by planetary density ($\rho$), rendering the traditional Earth Similarity Index (ESI) obsolete for primary filtering.
>
> **(Аннотация:** Мы представляем новый фундаментальный принцип астробиологии — **Закон Экзолоджики** (Принцип Критического Плотностного Окна). На основе анализа более 9800 экзопланет с помощью нашего гибридного ИИ-движка мы демонстрируем, что обитаемость строго ограничена плотностью планеты ($\rho$), что делает традиционный Индекс Подобия Земле (ESI) устаревшим для первичной фильтрации.)

##  Key Discovery / Ключевое Открытие

Our research proves that stable biospheres of Earth-type can only exist within a narrow **Critical Density Window**:
**(Наше исследование доказывает, что стабильные биосферы земного типа могут существовать только в узком «Критическом Окне Плотности»):**

$$4.8 \le \rho \le 7.8 \text{ g/cm}^3$$

*   **Optimum Density / Оптимальная Плотность:** $\approx 6.26 \text{ g/cm}^3$
*   **Lower Limit (< 4.8):** "Ocean Worlds" — No silicate contact, blocked geochemical cycles.
    *(Нижний предел (< 4.8): «Миры-Океаны» — отсутствие контакта с силикатами, блокировка геохимических циклов.)*
*   **Upper Limit (> 7.8):** "Iron Planets" — Stagnant lid tectonics, no magnetic dynamo.
    *(Верхний предел (> 7.8): «Железные Планеты» — тектоника застойной крышки, отсутствие магнитного динамо.)*

##  Dataset & Methodology / Датасет и Методология

This repository contains the validated dataset generated by the **ExoLogica AI** pipeline (v2.0).
**(Этот репозиторий содержит валидированный датасет, сгенерированный пайплайном ExoLogica AI (версия 2.0).)**

*   **Total Objects / Всего Объектов:** 9,838 exoplanets and substellar objects.
*   **Source Data / Исходные Данные:** Aggregated from NASA Exoplanet Archive, Kepler, TESS, K2 missions.
*   **AI Processing / Обработка ИИ:**
    *   Mass reconstruction for non-transiting planets using Bayesian inference.
        *(Реконструкция масс для нетранзитных планет с использованием байесовского вывода.)*
    *   Calculation of the **PRI (Planetary Retainability Index)** instead of ESI.
        *(Расчет индекса PRI (Индекс Удерживаемости Планеты) вместо ESI.)*
    *   Classification into types: Habitable Earth, Ocean World, Iron Planet, etc.
        *(Классификация по типам: Обитаемая Земля, Мир-Океан, Железная Планета и т.д.)*

### Files Included / Включенные Файлы:
1.  `ExoLogica_Export.csv`: Curated list of prime candidates with high reliability.
    *(Отфильтрованный список прайм-кандидатов с высокой надежностью.)*
2.  `ExoLogica_Export-s.csv`: Full global database (9,800+ objects) for statistical validation.
    *(Полная глобальная база данных (9800+ объектов) для статистической валидации.)*
